# Prime Surface Solutions Website

A modern, responsive website built for a pressure washing business in San Antonio, TX.

## Features

* Before & after image slider
* Mobile-friendly design
* Contact form integration
* Smooth scrolling and animations

## Tech Used

* HTML
* CSS
* JavaScript

## Purpose

This project was created to showcase front-end development skills and build a real-world business website.
